# Black-Scholes Option Pricing Model

A Python-based finance project that calculates European call and put option prices using the Black-Scholes model. The project also estimates implied volatility and visualizes how option prices change with volatility.

---

# Features

* Calculates European call and put option prices
* Estimates implied volatility from market prices
* Generates option price vs volatility plots
* Uses numerical methods for volatility estimation

---

# Technologies Used

* Python
* NumPy
* SciPy
* Matplotlib

---

# Concepts Used

* Black-Scholes Model
* Implied Volatility
* Option Pricing
* Numerical Optimization
* Quantitative Finance
